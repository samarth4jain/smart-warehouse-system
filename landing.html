Quick Setup Guide

1. Clone Repository

git clone https://github.com/samarth4jain/smart-warehouse-system.git

2. Install Dependencies

pip3 install fastapi uvicorn sqlalchemy python-multipart

3. Initialize Database

python3 create_minimal_db.py

4. Start Server

uvicorn backend.app.main:app --host 0.0.0.0 --port 8001

5. Access Dashboard

http://localhost:8001/dashboard

6. View Documentation

Check the DASHBOARD_STATUS.md file for detailed setup instructions and troubleshooting.
